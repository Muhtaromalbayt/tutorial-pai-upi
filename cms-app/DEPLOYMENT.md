# CMS - Tutorial PAI

Build directory for Cloudflare Pages deployment.

## Cloudflare Pages Settings

- **Root directory**: `cms-app`
- **Build command**: `npm run build`
- **Output directory**: `dist`
- **Node.js version**: 18

## Environment Variables (Set in Cloudflare Dashboard)

```
VITE_API_BASE_URL=https://tutorial-pai-upi.pages.dev
```

Update the URL above with your actual production website URL.
